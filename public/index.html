URL Shortener Microservice

A service that accepts a URL and returns a shortened url as a JSON response.

If the shortened URL is passed, the user is redirected to the target URL.

Example creation usage:

https://urlsmall.herokuapp.com/new/https://www.google.com

Example creation output:

{ "original_url":"https://www.google.com", "short_url":"https://urlsmall.herokuapp.com/1" }

Example usage:

https://urlsmall.herokuapp.com/1

Redirects to:

https://www.google.com

To see a list of shortened URLs:

https://urlsmall.herokuapp.com/list
